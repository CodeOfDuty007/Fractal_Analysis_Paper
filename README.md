# Fractal Analysis

Repository for extracting multifractal features from token surprise scores.

## Files
- `fractal_math.py` — fractal analysis utilities and main runner (produces `analysis_results.json`).
- `scorer.py` — optional script to compute token surprise scores (uses `torch`, `transformers`, `datasets`).
- `fractal_scores.json` — example input (array of score sequences).
- `analysis_results.json` — output generated by `fractal_math.py` (excluded from VCS by `.gitignore`).

## Requirements
Python 3.9+ recommended. Use a virtual environment.

## Quick start
```bash
cd "$(pwd)"
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
# Run analysis (reads fractal_scores.json)
python fractal_math.py
```

## Notes
- `scorer.py` requires `torch`, and may need a GPU-enabled `torch` build for larger models.
- `analysis_results.json` will be created in the repository root when `fractal_math.py` finishes.
